## Clover Tasks 🍀

Clover Tasks is a simple To Do App UI concept that I designed and plan to implement using React.js as part of my front-end learning journey.

This project focuses on creating a clean, modern, and fresh task management interface that helps users organize daily tasks, track progress, and stay productive. The design uses soft green tones and clover-inspired elements to create a friendly and calming user experience.

### Features

* Add new tasks
* Set due dates
* Track daily progress
* Responsive UI for desktop and mobile

### Purpose of this project

This project was created as:

* a UI design practice project
* a personal challenge to turn design into a real web app
* a learning project for React.js implementation

### Tech plan

* **Design:** Figma
* **Front-end:** React.js
* **Styling:** CSS
